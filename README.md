# Employee Management System using ServiceNow

## 📌 Project Overview

The Employee Management System is a ServiceNow application developed using **App Engine Studio** to automate employee onboarding and offboarding processes. The application simplifies employee lifecycle management by automating approvals, task assignments, email notifications, and record management.

The project helps organizations reduce manual work, improve operational efficiency, and provide a centralized platform for managing employee-related activities.

---

- Employee Record Management
- Employee Onboarding Request
- Employee Offboarding Request

- HR Task Automation
- IT Task Automation
- Security Task Automation
- Email Notifications
